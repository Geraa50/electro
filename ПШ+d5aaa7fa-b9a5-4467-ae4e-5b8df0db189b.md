    ### Кейс 2: Электрические цепи и схемы

    **Суть:** Игрок собирает электрическую цепь из компонентов, чтобы запитать потребитель.

    **Ключевые объекты:**

    |Объект|Узел Godot|Поведение|
|-|-|-|
|Источник питания|Node2D|Задаёт напряжение|
|Провод|Node2D + Line2D|Соединяет элементы, передаёт ток|
|Резистор|Node2D|Снижает напряжение (закон Ома)|
|Переключатель|Node2D|Включает/разрывает цепь|
|Потребитель|Node2D|Цель уровня — должен получить нужное напряжение|

    **Физика:** `V = I * R` (закон Ома). Цепь замкнута → ток течёт. Последовательное/параллельное соединение.

    **Подход:** Цепь — граф. Узлы — элементы, рёбра — провода. Обход графа для проверки замкнутости и вычисления тока.

    

- ТЗ (Техническое Задание) по проектированию уровней

    ## Проектирование уровней головоломки

    ### Принцип последовательного усложнения

    1. **Уровень-обучение** — один элемент, очевидное решение (игрок привыкает к управлению)

    2. **Уровень-закрепление** — два элемента, одно усложнение

    3. **Уровень-комбинация** — несколько элементов, нужно комбинировать

    4. **Уровень-вызов** — нестандартная комбинация, требует размышления

    5. **Финальный уровень** — все элементы, максимальная сложность

    ### Каждый уровень описывается

    - **Цель** — что нужно сделать

    - **Доступные объекты** — какие элементы можно размещать

    - **Фиксированные объекты** — какие элементы уже стоят (нельзя двигать)

    - **Условие победы** — когда уровень считается пройденным

    - **Подсказка** — текст для игрока
